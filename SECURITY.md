# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please report them privately:

1. **GitHub Security Advisory** (preferred): Go to the [Security tab](https://github.com/interactivecats/heyi.am/security/advisories/new) and create a private advisory.
2. **Email**: Send details to security@heyi.am.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

Reports will be acknowledged as soon as possible. Fixes will be developed privately before any public disclosure.

## Scope

- The `heyiam` CLI (`cli/`)
- The Phoenix backend (`heyi_am_umbrella/`)
- The `howdoyouvibe` package (`packages/howdoyouvibe/`)
