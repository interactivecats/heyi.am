# howdoyouvibe

[![npm version](https://img.shields.io/npm/v/howdoyouvibe)](https://www.npmjs.com/package/howdoyouvibe)

Find out how you vibe with AI coding tools. Scans your local sessions and generates a personality breakdown.

## Get Started

```bash
npx howdoyouvibe
```

Analyzes your AI coding sessions from Claude Code, Cursor, Codex, and Gemini CLI. Generates a vibe profile based on how you use AI — your patterns, preferences, and working style.

100% local analysis. Optional sharing to [howdoyouvibe.com](https://howdoyouvibe.com).
