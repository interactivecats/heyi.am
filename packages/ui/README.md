# @heyiam/ui

Shared visualization components for [heyi.am](https://heyi.am).

## Components

- **WorkTimeline** — Multi-agent fork/join session timeline with Bezier curves, wave grouping, and scroll-aware legend
- **DirectoryHeatmap** — File activity heatmap across sessions
- **GrowthChart** — Session activity over time
- **SessionOverlay** — Full session detail view with transcript, tools, and stats

## Usage

Internal package used by the heyi.am CLI dashboard and public portfolio pages. Not intended for standalone use.
