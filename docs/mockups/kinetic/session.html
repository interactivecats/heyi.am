Decisions

Q&A

Question
Why did you choose a discriminated union over single-table inheritance for transaction types?
Answer
STI would have left nullable columns everywhere -- transferTo only makes sense for transfers, recurringSchedule only for recurring. The union pattern keeps each type clean and the TypeScript compiler catches missing fields.
Question
How does the cascade delete work between Account and Transaction?
Answer
Deleting an account cascades to its transactions. Initially had it set to restrict, but that made account cleanup impossible without manual transaction deletion first.
